# capacitor-webauthn-ui

## Pre-Requsisite

```console
cd webauthn
git clone https://github.com/darkedges/capacitor-webauthn.git
cd capacitor-webauthn
npm run watch
```

## Run

```console
cd webauthn
git clone https://github.com/darkedges/capacitor-webauthn-ui
cd capacitor-webauthn-ui
ng serve
```
